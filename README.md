# FoodLogistic-Jan

Aquesta és la web corporativa de FoodLogistic, és una empresa capdavantera en la distribució i logística alimentària a nivell nacional. Arran de l'obertura de noves rutes i la contractació de més personal, el seu volum de dades i les seves necessitats de comunicació s'han multiplicat. dins el mòdul Projecte Intermodular.

Per la creació d'aquesta web, s'han utilitzat les següents tecnologies i eines:

- Gemnini 
- HTML5/CSS3
- Statcounter per l'analítica
